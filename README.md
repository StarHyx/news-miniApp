# wechat miniapp- bestnews
新闻阅读微信小程序 

## 说明

1.新闻数据部分API [数闻](https://fenfa.shuwen.com/)

2.新闻html用`web-view`进行解析，在模拟器上可以正常调试，真机调试时小程序必须与**已认证**的服务号或企业号绑定才可以使用
